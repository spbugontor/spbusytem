🔐 Password Admin

📋 Panduan Pemesanan

Cara Memesan:

  1. Tunggu waktu pembukaan
  2. Isi data lengkap
  3. Masukkan KK & NIK (16 digit)
  4. Klik "Pesan Sekarang"

Persyaratan:

  • Wajib bawa bukti KK
  • 1 pemesanan = 1 tabung
  • Tidak boleh data sama dalam 24 jam

⚙️ Panduan Admin

📊 Dashboard

  • Total Kuota: Jumlah tabung LPG yang tersedia hari ini
  • Pesanan: Jumlah pemesanan yang sudah masuk
  • Sisa Stok: Kuota dikurangi pesanan (auto-update)
  • Sudah Bayar: Jumlah pemesan yang sudah mengkonfirmasi pembayaran
  • Belum Bayar: Jumlah pemesan yang belum melunasi
  • Total Nominal: Pendapatan dari pemesan yang sudah bayar (harga × jumlah)
  • ⚠️ Daftar Belum Bayar: List real-time pemesan yang belum lunas

📋 List Pemesan

  • Cari Nama: Gunakan kolom pencarian untuk menemukan pemesan spesifik
  • Tandai Bayar: Klik tombol "Tandai Bayar" untuk konfirmasi pembayaran pemesan
  • Verifikasi: Sistem akan menampilkan modal verifikasi dengan data pemesan sebelum update
  • Hapus: Klik tombol "Hapus" untuk menghapus pesanan dari list
  • Status Badge: Hijau (Lunas) atau Kuning (Belum Bayar)

⚙️ Pengaturan

  • Tanggal Buka: Pilih tanggal dimulainya pemesanan
  • Jam Buka: Atur jam tepat pembukaan pemesanan (HH:MM)
  • Kuota LPG: Total tabung LPG yang disediakan hari ini
  • Harga/Tabung: Harga per tabung (format Rp tanpa titik)
  • Maks. Tabung per KK/Hari: Batas pemesanan per nomor KK dalam 1 hari
  • Simpan Pengaturan: Klik untuk menyimpan semua perubahan
  • Reset Semua Data: Tombol untuk menghapus seluruh pesanan (hati-hati!)

🔐 Password

  • Password Lama: Masukkan password admin yang saat ini aktif
  • Password Baru: Buat password baru minimal 4 karakter
  • Konfirmasi: Ulangi password baru untuk memastikan kecocokan
  • Ubah Password: Klik untuk update password
  • ⚠️ Simpan password baru di tempat aman!

💡 Tips Penggunaan

  • ✓ Atur pengaturan SEBELUM pembukaan pemesanan
  • ✓ Monitor dashboard untuk melihat status real-time
  • ✓ Update status pembayaran segera saat pemesan membayar
  • ✓ Gunakan fitur reset hanya saat perlu ulang dari awal
